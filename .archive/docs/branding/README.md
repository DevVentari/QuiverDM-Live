# QuiverDM Brand Identity

This folder contains the comprehensive brand identity system for QuiverDM, an AI-powered D&D session management tool for Dungeon Masters.

## Quick Reference

### Primary Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Brand Purple | `#8B5CF6` | Primary brand color, CTAs, links |
| Gold Accent | `#d4a84b` | Warm accents, highlights, premium elements |
| Cream Text | `#e8dcc8` | Primary text on dark backgrounds |
| Dark Brown | `#0f0d0b` | Primary background |

### Typography

- **Display Font**: Cinzel (headings, titles)
- **Body Font**: Crimson Text (paragraphs, UI text)

### Brand Personality

- **Fantasy-meets-modern**: D&D heritage with clean tech sensibility
- **Organized & reliable**: A trustworthy tool for DMs
- **Warm & inviting**: Medieval tavern atmosphere, not cold corporate
- **Empowering**: Helps DMs be their best

## Workflow

1. **Explore** - Review logo concepts in `logo-exploration.md`
2. **Generate** - Use AI prompts to create logo concepts
3. **Select** - Choose winning direction
4. **Refine** - Iterate on chosen style
5. **Document** - Complete remaining brand guidelines
6. **Implement** - Generate actual assets for `public/`
