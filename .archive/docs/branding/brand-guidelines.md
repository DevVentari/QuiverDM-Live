# Brand Guidelines

Comprehensive rules for using the QuiverDM brand identity consistently.

---

## Brand Story

### Who We Are

QuiverDM is an AI-powered session management tool designed specifically for Dungeon Masters. Like a well-stocked quiver gives an archer confidence in battle, QuiverDM gives DMs the tools they need to run memorable sessions.

### Brand Personality

| Trait | Description | Voice Example |
|-------|-------------|---------------|
| **Organized** | We bring order to campaign chaos | "Everything you need, right where you need it" |
| **Empowering** | We make DMs more capable | "Run your best session ever" |
| **Warm** | We're approachable, not intimidating | "Welcome back to the tavern" |
| **Knowledgeable** | We understand D&D deeply | "Built by DMs, for DMs" |
| **Modern** | We use tech to solve real problems | "AI that actually helps" |

### Brand Promise

*"QuiverDM helps Dungeon Masters organize their campaigns and run better sessions through intelligent tools that understand tabletop RPGs."*

## Logo Usage

### Primary Logo

The QuiverDM logo should be used consistently across all applications.

**Preferred Usage**:
- Full color logo on dark backgrounds
- Icon-only for small applications (< 48px)
- Maintain clear space equal to the height of the "Q"

### Minimum Sizes

| Format | Minimum Width |
|--------|---------------|
| Full logo | 120px |
| Icon only | 16px |
| Wordmark | 80px |

### Clear Space

*[Diagram showing clear space requirements]*

The logo requires breathing room equal to the height of the "Q" letterform on all sides. No other elements should intrude into this space.

### Backgrounds

**Approved**:
- Dark backgrounds (`#0f0d0b`, `#1a1714`)
- Neutral dark grays
- Purple gradient backgrounds (brand purple family)
- Clean white/cream backgrounds (use dark logo version)

**Avoid**:
- Busy photographic backgrounds
- Competing brand colors
- Low-contrast combinations

---

## Logo Don'ts

### Never Do These

1. **Don't stretch or distort** - Maintain original proportions
2. **Don't change colors** - Use only approved color variations
3. **Don't add effects** - No shadows, glows, or 3D effects
4. **Don't rotate** - Keep logo horizontal
5. **Don't crop** - Show complete logo
6. **Don't outline** - Don't add strokes around the logo
7. **Don't animate** - Logo should remain static (except loading states)
8. **Don't recreate** - Always use official logo files

## Co-Branding

### Partner Logos

When displayed alongside partner logos:
- QuiverDM logo should be equal or larger in size
- Maintain clear space between logos
- Use consistent color treatment (all full color or all monochrome)

### "Powered by" Usage

When QuiverDM is integrated into other products:
```
Powered by QuiverDM
```
Use the icon-only logo at 24-32px alongside the text.
